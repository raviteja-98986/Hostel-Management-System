

🚀 **Built a Hostel Management System using Django**

I recently designed and developed a **Hostel Management System** that helps manage room availability and hostel members efficiently.

🔹 **Key Features I Implemented**

🏨 **Visual Room Availability**

* Rooms are displayed as cards with color indicators:
* 🔴 **Red** → Room is full
* 🟡 **Yellow** → Room is partially filled
* 🟢 **Green** → Room is fully available
  This makes it very easy for the admin to quickly identify room status.

🏢 **Floor-Based Room Organization**

* Rooms are organized by **floor number**, making navigation and management simple.

👤 **Admin Member Management**

* Admin can **add members to rooms**
* Admin can **edit member details**
* Admin can **delete members from rooms**

🔄 **Smart Room Shift Feature (Main Feature)**

* Admin can click **“Change Room”** for a member
* The system automatically shows **only available rooms**
* Displays **sharing capacity and available beds**
* Allows the admin to **shift a member to another room without manually re-entering details**

🔐 **Two Types of Login**

* **Admin Login** → Full control to manage rooms and members
* **User Login** → Can view hostel information and room availability

* Database relationships and room occupancy logic
* Building **real-world management systems**
* Designing intuitive **admin workflows**
